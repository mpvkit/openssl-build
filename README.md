build scripts for [openssl](https://github.com/openssl/openssl)

## Installation

### Swift Package Manager

```
https://github.com/mpvkit/openssl-build.git
```

## How to build

```bash
make build
# or build specified platforms 
make build platform=ios,macos
# or see help
make help
```
